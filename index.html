Sistema de Agricultura Inteligente

O código é um sistema que coleta dados do usuário sobre agricultura, como tipo de vegetal, solo e clima. Ele armazena essas informações em um banco de dados SQLite e fornece previsões de produtividade usando IA. O sistema também oferece recomendações personalizadas com base nas escolhas feitas pelo usuário..

SmartConnect

O projeto é uma implementação de Internet das Coisas (IoT) que envolve comunicação e controle de dispositivos. Ele inclui scripts, bancos de dados e integrações com plataformas como Wokwi, pode ser testado ou simulado em um ambiente virtual.
